## Init

- Clone this repo [ssh or https]
```
git clone git@github.com:WailanTirajoh/MyFirstNuxtBackend.git
```
- Copy .env.example to .env
```
cp .env.example .env
```
- create database [mysql or pgsql]
```
mysql -u <username> -p
create database my_first_app;
```
- migrate
```
php artisan migrate
```
- serve
```
php artisan serv
```
